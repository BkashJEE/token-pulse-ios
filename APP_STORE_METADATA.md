# App Store metadata draft

## Name

Token Pulse: AI Usage Monitor

## Subtitle

Private agent usage and quota status

## Description

Token Pulse gives you a private, read-only view of your AI runtime activity while you are away from your computer. Monitor normalized token usage, active sessions, provider quota remaining, and local hardware readiness from your own Token Pulse relay.

The app supports self-hosted Token Pulse servers. Access keys are stored in the iOS Keychain and raw prompts, messages, and desktop credentials are not requested by the mobile app.

## Keywords

AI, tokens, Codex, agents, quota, usage, developer tools, monitoring

## Review notes draft

The app is a read-only client for a user-configured Token Pulse server. Provide App Review with a dedicated demo server URL and temporary read key. No desktop write credential is entered or stored in the iOS app.
